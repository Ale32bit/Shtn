# Shtn
Node.js fast URL  Shortener


## Config

```json
{
  "url":"https://shtn.ga", // URL
  "name":"Shtn.ga", // Title
  "motto":"URL Shortener", // Motto
  "port":80, // Port to listen to
  "sessionSecret":"SECRET HERE", // Just put a random long string and you're OK
  "sql":{
    "host":"DB HOST", // Database Host
    "user":"USERNAME", // Database User
    "password":"PASSWORD", // User password
    "db":"DB NAME" // Database name
  }
}
```
